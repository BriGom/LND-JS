## 🔴 Nivel avanzado (consolidación de bucles y lógica)
### 1️⃣1️⃣ FizzBuzz (El clásico)
Usa un bucle para recorrer los números del 1 al 20.

1. Si el número es múltiplo de 3, muestra "Fizz".
2. Si es múltiplo de 5, muestra "Buzz".
3. Si es múltiplo de 3 y de 5, muestra "FizzBuzz".
4. Si no cumple ninguna, muestra el número.

### 1️⃣2️⃣ Invertir una palabra
Declara una variable con una palabra corta.
Usa un bucle para leer la palabra de atrás hacia adelante y armar una nueva cadena.
Muestra la palabra invertida en consola.

### 1️⃣3️⃣ Factorial de un número
Declara una variable con un número entero positivo (ej. 5).
Calcula su factorial multiplicando todos los números del 1 hasta ese número (1 * 2 * 3 * 4 * 5).
Muestra el resultado final.

## 🟣 Nivel extra (condicionales complejos y patrones)
### 1️⃣4️⃣ Piedra, papel o tijera
Declara dos variables: jugador1 y jugador2 (asígnales "piedra", "papel" o "tijera").
Usa condicionales para determinar quién gana o si hay empate.
Muestra el ganador en consola.

### 1️⃣5️⃣ Días del mes
Declara una variable con el nombre de un mes en minúsculas.
Usa un switch para mostrar en consola cuántos días tiene ese mes (asume que febrero tiene 28 días).

### 1️⃣6️⃣ Triángulo de asteriscos

Usa un bucle para imprimir en consola un patrón escalonado de 5 niveles:

```
*
**
***
****
*****
```
